# Demo Dataset for ComplianceOS

This folder contains realistic sample evidence files for testing and demonstrating ComplianceOS.

## Purpose

These files are used for:
1. **Regression testing** - Verify the system works correctly
2. **Demo stability** - Consistent demonstration environment
3. **Verification** - Validate end-to-end golden path

## Important Notes

- **No hardcoded outputs** - All analysis is performed by the AI in real-time
- **Real framework definitions** - Files are analyzed against actual PIPA/Labor Law requirements
- **Traceability verified** - AI must cite specific excerpts from these documents

## Sample Files

### Privacy Domain (개인정보 보호)

1. **privacy_policy_sample.txt** - Sample privacy policy document
2. **consent_form_sample.txt** - Sample consent collection form
3. **data_processing_agreement_sample.txt** - Sample vendor DPA

### Labor Domain (근로기준법)

4. **working_hours_record_sample.txt** - Sample time tracking record
5. **overtime_approval_sample.txt** - Sample overtime pre-approval form

## Usage

1. Start the application
2. Complete onboarding with a demo company profile
3. Upload these files as evidence
4. Link to appropriate evidence requirements
5. Verify AI analysis produces traceable, accurate results
